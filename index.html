Leidy Achury - Cotizador de Proyectos de Software y Hardware



Este simulador es un prototipo de cotizador para proyectos de software o hardware en la organización. Permite ingresar información básica de un proyecto y generar un resumen en consola y en cuadros de diálogo, facilitando la simulación de un proceso de presupuestación.

La finalidad del simulador es brindar una herramienta didáctica que represente de forma sencilla cómo se gestionan los presupuestos de proyectos tecnológicos en WideTech - Empresa que diseña soluciones de software basados en Servicios de localización. El diseño combina HTML para la interfaz y JavaScript para la lógica de captura y validación de datos.
